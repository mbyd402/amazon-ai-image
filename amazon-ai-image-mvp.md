# 亚马逊 AI 商品图工具站 - MVP 详细设计

## 一、MVP 核心目标

- 验证 **产品定位**：亚马逊卖家是否愿意为"一站式AI图片处理工具"付费
- 验证 **核心功能**：四个核心功能的用户满意度
- 验证 **付费转化**：免费用户到付费用户的转化率
- 最小化开发，最快上线

## 二、MVP 功能范围

✅ **必须包含的核心功能**

### 1. 用户系统
- [ ] 邮箱注册/登录（支持 Google 一键登录更好，MVP 可以先做邮箱）
- [ ] 用户个人中心
- [ ] 查看处理历史
- [ ] 查看套餐用量/额度

### 2. 核心功能模块（四个核心）

| 功能 | MVP 实现范围 | 技术方案建议 |
|------|-------------|--------------|
| **AI 纯白背景** | 上传图片 → AI 自动抠图 → 替换为亚马逊标准纯白（RGB 255,255,255）→ 下载 | 先用 rembg 开源模型，或调用 Remove.bg API 快速验证 |
| **AI 去水印/文字/Logo** | 用户框选需要移除的区域 → AI 自动修复 → 下载 | 采用 LaMa 修复模型，或调用第三方 AI 修复 API |
| **图片高清增强 + 画质修复** | 1x/2x/4x 超分辨率放大 → 降噪+锐化 → 下载 | 开源模型用 Real-ESRGAN，或第三方 Upscale API |
| **亚马逊主图合规检查** | 上传图片 → AI 检测以下项 → 给出合规报告：<br>1. 背景是否纯白<br>2. 是否包含文字/水印<br>3. 是否包含促销信息<br>4. 尺寸是否符合要求（≥1000px）<br>5. 图片比例是否正确 | 用 CV 模型做对象/文字检测，规则+AI结合，不用太复杂，先覆盖主要违规点 |

### 3. 基础批量处理（MVP 简化版）
- 支持一次上传最多 10 张图（避免算力压力）
- 批量应用同一操作（比如全部做纯白背景）
- 批量下载 ZIP 包

### 4. 支付与订阅（调整为套餐包模式，适配 PayPal 中国个人）

**改为套餐包模式，没有月订阅，降低风控风险：**

| 套餐 | 价格 | 张数 | 说明 |
|------|------|------|------|
| 免费 | $0 | 3 张 | 体验 |
| 迷你 | $4.99 | 15 张 |  |
| 标准 | $14.99 | 50 张 |  |
| 无限 | $29.99 | 200张/30天 |  |

- 点数永久有效，用户没压力
- 都是一次性交易，PayPal 风控更安全
- 以后验证成功再加上月订阅选项

### 5. 基础营销页面
- 首页：功能介绍 + 定价 + 在线演示
- 关于页
- 帮助页（简单教程）
- 隐私政策 + Terms of Service（合规必备）

❌ **MVP 不做（后续迭代）**
- 移动 App（先做 Web 响应式，能在手机用就行）
- AI 生成场景背景
- API 对接亚马逊/Shopify
- A+ 模板设计
- 社区/论坛
- 多语言（先英文，满足美国市场就行）

## 三、技术架构建议

### 前端
- 框架: Next.js 14 + React
- 样式: TailwindCSS（开发快，美观）
- 部署: Vercel（免费起步，方便）
- 响应式: 支持桌面/移动，卖家手机也能处理

### 后端
- 框架: Next.js API Routes
- 数据库: PostgreSQL（Supabase 免费起步）
- 存储: Cloudflare R2（R2 免费流量多）
- AI 模型部署:
  - **方案 A（最快）**: 调用第三方 API（Remove.bg, Clipdrop 等），不用自己部署模型，最快上线 ✓
  - **方案 B（性价比高）**: 开源模型部署在 Fly.io / Render，自己控制成本
- **MVP 推荐方案 A**，验证后再换方案 B

### 支付
- PayPal（适配中国个人账号，一次性交易更安全）

### 部署
- 前端: Vercel
- 后端: 同前端
- 数据库: Supabase
- 存储: Cloudflare R2

## 六、MVP 上线后需要验证的核心问题

1. **获客**：从卖家社区获取 100 个注册用户需要多少成本？
2. **激活**：用户注册后，是否真的使用核心功能？至少完成一次处理？
3. **留存**：用户会不会次月再来使用？
4. **付费**：转化率能到多少？（行业平均 2-5% 就很好）
5. **口碑**：用户最满意哪几个功能？最不满意什么？

## 八、MVP 核心变化（从原计划调整）

### 1. 支付模式调整 → 适配 PayPal 中国个人
- 改为套餐包模式，没有月订阅，降低风控风险
- 点数永久有效，用户没压力
- 都是一次性交易，PayPal 风控更安全
- 以后验证成功再加上月订阅选项

### 2. 技术方案 → 第三方 API 快速起步
不用自己部署 AI 模型，直接对接现成 API：
- 抠图：Remove.bg → $2 / 100 张
- 去水印：Clipdrop Inpaint → $0.025 / 张
- 高清放大：Clipdrop Upscale → ~$0.02 / 张
- 合规检查：Cloudmersive → 免费额度起步

**单张成本：$0.02-$0.05，你卖 $0.33/张 起，毛利 85%+**

### 3. 启动总成本 → 不到 200 人民币
- 域名：$15/年
- AI 预充值：$10-$20
- 其他全免费（Vercel + Supabase + R2）

### 4. 开发时间 → 还是 2-3 周上线
